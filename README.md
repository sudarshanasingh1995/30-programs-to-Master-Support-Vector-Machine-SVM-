Welcome to the ultimate hands-on repository for mastering **Support Vector Machines (SVM)**! This repository contains a curated collection of **30 practical Python programs** engineered to take you from foundational intuition to production-ready implementation. 


SVM remains one of the most mathematically elegant and robust supervised learning algorithms in data science. This project bridges the gap between complex mathematical theory and practical execution using clean, well-documented code.

---

## 🗺️ Learning Roadmap

The 30 programs are structured progressively across three major phases:

### 🧩 Phase 1: The Fundamentals (Programs 1–10)
* **Geometric Intuition:** Understand how data points are separated in n-dimensional space.
* **Hyperplanes & Margins:** Learn the mathematical differences between Hard and Soft margins.
* **Support Vectors:** Isolate and visualize the critical data points that dictate the decision boundary.

### ⚡ Phase 2: The Kernel Trick (Programs 11–20)
* **Non-Linear Classification:** Master how to project low-dimensional data into higher dimensions without the computational tax.
* **Kernel Deep-Dive:** Hands-on implementations using:
  * Linear Kernel
  * Polynomial Kernel
  * Radial Basis Function (RBF) Kernel
  * Sigmoid Kernel

### 🛠️ Phase 3: Advanced Applications & Optimization (Programs 21–30)
* **Regression (SVR):** Learn how to apply SVM mechanics to continuous numerical predictions.
* **Hyperparameter Tuning:** Master the optimization of regularizers ($C$), gamma ($\gamma$), and epsilon ($\epsilon$).
* **Real-world Pipelines:** Deploy automated `GridSearchCV` pipelines, handle imbalanced datasets, and evaluate models using precision-recall metrics.

---

## 💻 Tech Stack & Key Features

* **Language:** Python 3.x
* **Core Libraries:** `scikit-learn`, `NumPy`, `pandas`, `matplotlib`, `seaborn`
* **Clean Code:** Every script is self-contained, commented, and structured for quick scannability.
* **Visualizations:** Dedicated scripts for plotting 2D and 3D decision boundaries so you can *see* how the algorithm learns.
